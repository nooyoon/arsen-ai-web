# 공개 로드맵 링크 반영 제안

작성일: 2026-05-09
상태: 홈페이지 반영 전 검토용 초안 (index.html 미수정)
관련 자원:
- 대상 페이지: `index.html`
- 연결 대상: `status/roadmap.html` (공개 게시 시 `/status/roadmap.html`)
- 보조 자료: `status/implementation-roadmap.json`

## 목적

`status/roadmap.html` 가 이미 공개 디렉토리에 존재하지만 홈페이지(`index.html`)에서 진입 경로가 없습니다. 방문자가 ARSEN AI 의 진행 상황을 확인할 수 있도록 진입 동선을 추가할지 검토합니다.

본 문서는 제안만 담고 있으며, `index.html` 은 수정하지 않았습니다. 반영 여부는 운영자 승인 후 결정합니다.

## 권장 위치 (3안)

### 안 A. 네비게이션 바에 "준비 현황" 추가 (권장)

- 위치: `nav-links` 영역, 기존 항목 사이 또는 우측 GitHub 버튼 직전
- 라벨 후보: `준비 현황`, `로드맵`, `Status`
- 장점: 페이지 진입 직후 노출, 의도적으로 찾는 사용자 동선 단순화
- 단점: 네비게이션 항목이 1개 늘어 모바일 폭에서 메뉴 항목 재정렬 필요

추가 위치 후보:
```
프로젝트 / 에이전트 / 채널 / [준비 현황] / 연락처 / GitHub →
```

### 안 B. 푸터의 보조 링크로 추가

- 위치: `footer-links` 영역 (현재 GitHub, YouTube 노출 중)
- 라벨 후보: `준비 현황`, `Status`
- 장점: 홈페이지 헤더 디자인 영향 최소, 안정적
- 단점: 노출 강도 낮음, 신규 방문자 발견 가능성 떨어짐

### 안 C. 히어로 섹션 보조 버튼으로 추가

- 위치: `.hero-btns` 영역, "프로젝트 보기" / "AI 모임 신청" 옆
- 라벨 후보: `📊 준비 현황 보기`
- 장점: 첫 화면에서 직접 노출, 방문자가 "지금 어디까지 왔는가" 확인 가능
- 단점: 모집 CTA 와 시각 경쟁, 버튼 3개 시 모바일 레이아웃 영향

권장 조합: **안 A + 안 B 동시 적용** (네비게이션 + 푸터 양쪽 노출). 안 C 는 모집 CTA 동선 보호를 위해 보류 권장.

## 반영 시 점검 항목 (운영자 승인 필요)

다음 항목은 실제 `index.html` 수정 전에 결정이 필요합니다.

1. 라벨 확정: `준비 현황` / `로드맵` / `Status` 중 1개
2. 적용 범위: 네비게이션 / 푸터 / 히어로 중 어디까지
3. 링크 경로 형식: 상대 경로 `status/roadmap.html` vs 절대 경로 `/status/roadmap.html`
   - 권장: 절대 경로 `/status/roadmap.html` (서브 페이지에서도 일관 동작)
4. `target="_blank"` 사용 여부: 권장 안은 동일 탭 (내부 페이지)
5. 모바일 네비게이션 메뉴 동기 반영 여부 (`#navLinks.open` 영역)
6. 접근성: 링크 텍스트만으로 의미 전달 가능한지 확인 (아이콘만 사용 금지)

## 적용 예시 (참고용 – 미적용)

다음 스니펫은 안 A 채택 시 `nav-links` 영역에 들어갈 형태의 예시입니다. 본 문서는 제안 단계이며 실제 적용은 운영자 승인 후에 별도 작업으로 진행합니다.

```
<a href="/status/roadmap.html">준비 현황</a>
```

푸터 영역(안 B) 적용 시 예시:

```
<a href="/status/roadmap.html">📊 준비 현황</a>
```

## 공개 안전성 검사

본 제안서가 외부에 노출되어도 문제없는지 점검한 결과입니다.

- 내부 경로 노출: 없음 (홈 디렉토리, 사용자 계정명, 머신명 미포함)
- 비밀값/토큰: 없음 (.env, API key, cookie 미포함)
- 내부 시스템 식별자: 없음 (private 레포명, IP, 호스트명 미포함)
- 외부 링크: 기존 `index.html` 에 이미 공개된 도메인(`arsen-ai.com`, `apply.arsen-ai.com`, GitHub `nooyoon`, YouTube `@Arsene-ai`)만 언급
- 미공개 기획/내부 일정: 없음 (반영 위치 후보와 라벨 후보만 다룸)
